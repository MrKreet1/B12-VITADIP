# Расчетное исследование устойчивых геометрий нейтрального кластера B12

## Аннотация

В работе выполнен расчетный поиск низкоэнергетических геометрий нейтрального кластера B12 методом последовательного двухэтапного скрининга. На первом этапе оптимизированы 78 стартовых конфигураций на уровне `BP86/def2-SVP` с дисперсионной поправкой D3BJ. На втором этапе пять наиболее низкоэнергетических кандидатов пересчитаны на уровне `B3LYP/def2-TZVP` с использованием `RIJCOSX`, D3BJ, оптимизации геометрии и частотного анализа. По результатам уточненного расчета минимальной энергией обладает квазипланарная структура `rank02_quasi_planar_nn2.00_perturb0` с полной энергией `-297.87422668` Hartree. Частотный анализ не выявил мнимых частот; первая колебательная частота составила `199.90 cm**-1`. Полученные данные указывают на квазипланарный мотив как наиболее устойчивый среди рассмотренного набора стартовых конфигураций B12.

Ключевые слова: B12, кластер бора, DFT, ORCA, геометрическая оптимизация, частотный анализ, квазипланарная структура.

## 1. Введение

Кластеры бора характеризуются выраженным многоцентровым связыванием и высокой чувствительностью относительной устойчивости изомеров к геометрии. Для нейтрального B12 особенно важно рассматривать не одну стартовую структуру, а набор существенно различных топологических мотивов, поскольку локальная оптимизация методом DFT может приводить к разным локальным минимумам в зависимости от начальных координат.

Цель настоящей работы состояла в выявлении наиболее устойчивого кандидата на минимум нейтрального B12 среди набора стартовых конфигураций, включающего компактные, квазипланарные, кольцевые, случайные и высокосимметричные мотивы. Дополнительная задача заключалась в проверке выбранных низкоэнергетических структур более строгим расчетом с частотным анализом, позволяющим отличить минимум потенциальной поверхности от седловой точки.

## 2. Объект и исходные данные

Объект исследования: нейтральный кластер B12 в синглетном состоянии, заряд `0`, мультиплетность `1`.

Исходный набор включал 78 XYZ-конфигураций из директории `configs/`. Набор содержит следующие группы структур:

- регулярные топологии при нескольких масштабах межатомных расстояний;
- возмущенные варианты выбранных регулярных топологий;
- случайные стартовые конфигурации;
- контрольные растянутые конфигурации с характерным расстоянием `5.0` Angstrom.

Основные файлы, использованные при подготовке отчета:

| Файл / директория | Содержание |
|---|---|
| `configs/` | стартовые координаты |
| `jobs/` | первичные ORCA-расчеты |
| `jobs_refined/` | уточненные расчеты пяти лучших кандидатов |
| `results.csv` | итоговая таблица первичного скрининга |
| `results_refined.csv` | итоговая таблица уточненного этапа |
| `best_structure.xyz` | лучший кандидат первичного этапа |
| `best_refined.xyz` | лучший кандидат после уточнения |

## 3. Методика расчета

### 3.1. Первичный скрининг

Первичный скрининг проведен на уровне:

```text
BP86 / def2-SVP / def2/J / RI / D3BJ / Opt / TightSCF
```

Ключевые параметры ORCA-входов:

```text
! BP86 def2-SVP def2/J RI Opt TightSCF D3BJ
! SlowConv
%pal nprocs 8 end
%maxcore 2500
%geom MaxIter 300 end
* xyz 0 1
```

Задача данного этапа заключалась в широком, но вычислительно доступном поиске низкоэнергетических областей поверхности потенциальной энергии. Расчеты запускались последовательно, каждая задача использовала 8 ядер.

### 3.2. Уточнение низкоэнергетических кандидатов

Пять лучших структур первичного этапа были пересчитаны на более высоком уровне:

```text
B3LYP / def2-TZVP / def2/J / RIJCOSX / D3BJ / Opt + Freq
```

Ключевые параметры ORCA-входов:

```text
! B3LYP def2-TZVP def2/J RIJCOSX Opt TightSCF D3BJ Freq
%pal nprocs 8 end
%maxcore 2500
%geom MaxIter 500 end
* xyz 0 1
```

Наличие ключевого слова `Freq` позволило провести гармонический частотный анализ. В качестве критерия локального минимума использовалось отсутствие мнимых частот после исключения шести нулевых мод, соответствующих поступательным и вращательным степеням свободы нелинейной молекулярной системы.

### 3.3. Обработка результатов

Энергии извлекались из ORCA output-файлов по последнему значению `FINAL SINGLE POINT ENERGY`. Успешным считался расчет, удовлетворяющий условиям:

- наличие нормального завершения ORCA;
- сходимость геометрической оптимизации;
- наличие итоговой энергии;
- наличие оптимизированного XYZ-файла.

Относительные энергии рассчитаны относительно минимальной энергии в соответствующем этапе:

```text
dE = E_i - E_min
```

Для перевода единиц использованы коэффициенты из `analyze_results.py`:

```text
1 Hartree = 27.211386245988 eV
1 Hartree = 627.5094740631 kcal/mol
```

## 5. Обсуждение

Полученные результаты показывают, что выбранный протокол поиска устойчиво приводит к квазипланарным геометриям как наиболее низкоэнергетическим структурам нейтрального B12. На первичном уровне `BP86/def2-SVP` несколько квазипланарных стартовых конфигураций сходятся к практически одинаковым энергиям. Это указывает либо на достижение очень близких геометрий, либо на наличие нескольких близких минимумов в пределах точности данного уровня теории.

Уточнение на уровне `B3LYP/def2-TZVP` изменило порядок первых двух кандидатов: структура `quasi_planar_nn2.00_perturb0`, занимавшая второе место в первичном скрининге, стала наиболее низкоэнергетической. Однако энергетический разрыв между первым и вторым уточненными кандидатами составляет всего `0.001` kcal/mol, что существенно меньше типичной погрешности DFT-подхода. Поэтому различие между ними не следует интерпретировать как надежное разделение двух различных химических состояний без дополнительного анализа геометрического совпадения, ZPE-поправок и, при необходимости, расчетов на альтернативных функционалах.

Икосаэдрические, кубооктаэдрические, призматические и кольцевые мотивы в данном наборе оказались существенно выше по энергии. Особенно показательно положение икосаэдрического семейства: лучший икосаэдрический кандидат выше первичного минимума на `3.1467` eV (`72.564` kcal/mol). Это согласуется с выводом о предпочтительности менее объемной, квазипланарной организации атомов бора в нейтральном B12.

## 6. Ограничения исследования

Настоящие результаты следует рассматривать как расчетный вывод в рамках выбранного протокола. Основные ограничения:

- уточнение проведено только для пяти лучших структур первичного скрининга;
- не выполнено сравнение с триплетным состоянием;
- в итоговой таблице не учтены ZPE и термохимические поправки;
- не выполнена проверка устойчивости порядка изомеров на других функционалах;
- не проведено автоматическое сопоставление геометрий для выявления дублирующих минимумов.

Наиболее важное практическое ограничение: структура `trunc_tetrahedron_nn2.00` в первичном скрининге находится всего на `0.0003` eV выше минимума. Для строгого академического заключения рекомендуется включить ее в следующий уточненный расчет вместе с топ-10 или топ-12 первичного списка.

## 7. Заключение

Выполнен двухэтапный расчетный скрининг геометрий нейтрального B12. Из 78 стартовых конфигураций 74 успешно оптимизированы на первичном уровне `BP86/def2-SVP`. Уточнение пяти лучших кандидатов на уровне `B3LYP/def2-TZVP` с частотным анализом показало, что минимальную энергию имеет квазипланарная структура `rank02_quasi_planar_nn2.00_perturb0`.

Основной результат:

```text
E = -297.87422668 Hartree
dE = 0.0000 eV
первая колебательная частота = 199.90 cm**-1
мнимые частоты не обнаружены
```

Таким образом, в рамках проведенного исследования наиболее устойчивым кандидатом на минимум нейтрального B12 является квазипланарная структура, сохраненная в файле `best_refined.xyz`.
